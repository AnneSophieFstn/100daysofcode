# Todo List React Native

## Description

Cette application est une todo list simple et intuitif développé en React Native avec TypeScript. Elle permet de suivre les tâches qu'on à a faire avec des fonctionnalités comme ajouter, supprimer, valider une tâche. L'interface utilisateur est claire et adaptée aux écrans mobiles.

## Fonctionnalités

- Afficher les tâches.
- Ajouter une tâche.
- Valider une tâche.
- Supprimer une tâche.

## Installation

1. Clonez le dépôt :

   ```bash
   git clone
   cd day2

   ```

2. Installez les dépendances :
   npm install

3. Lancez l'application

   - Sur Expo:
     npm start
   - Pour Android:
     npm android

   - Pour IOS:
     npm ios

## Technologies utilisées

- [React Native](https://reactnative.dev/) : Framework pour développer des applications mobiles.
- [TypeScript](https://www.typescriptlang.org/) : Superset de JavaScript pour un typage fort.
